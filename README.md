# 经典DOA估计算法
This project is a study record of the millimeter wave radar classic DOA estimation algorithm.

本项目为毫米波雷达经典DOA估计算法学习记录。

## 项目文档构成

`xxx_sim.m`为绘图程序。

`DOA_xxx.m`为算法程序。
